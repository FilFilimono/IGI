Как заменить фотографии на свои
================================

1. Скачайте картинки (мебель, портреты сотрудников) в формате jpg или png.

2. Положите файлы в папки:
   static/seed/furniture/   — для каталога
   static/seed/employees/   — для контактов (фото сотрудников)

3. Имена файлов (любой вариант):
   - по коду товара: KU-001.jpg, SP-002.png
   - по порядку: 1.jpg, 2.jpg, 3.jpg (пойдут по списку в БД)
   - для сотрудников: 1.jpg, 2.jpg или ivanov_dmitriy.jpg

4. В корне проекта выполните:
   python manage.py seed_data --replace-images

5. Перезапустите сервер и обновите страницу в браузере.

Можно также загрузить фото через админ-панель: /admin/
